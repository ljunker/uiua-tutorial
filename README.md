# Practical Uiua

A Quarkdown tutorial for learning [Uiua](https://www.uiua.org/) through practical exercises.

Read the rendered tutorial at <https://ljunker.github.io/uiua-tutorial/>.

The guided path currently has 10 steps, plus a separate glyph curriculum ordered by difficulty.

## Requirements

- `uiua` installed via Cargo
- `quarkdown`

The project was created against `uiua 0.18.1` and `quarkdown 2.0.0`.

## Build

```sh
make build
```

The rendered HTML is written to `quarkdown-output/Practical-Uiua/`.

## Check

```sh
make check
```

This checks all Uiua example files and compiles the Quarkdown documentation.

## Learn

Start with `main.qd`, or open the rendered `quarkdown-output/Practical-Uiua/index.html` after building.
